# Web Frontend — Public Restaurant Website

**Technology**: React, TypeScript, Vite, Nginx (prod)
**Entry Point**: `index.tsx` → `App.tsx`
**Parent Context**: [../CLAUDE.md](../CLAUDE.md)

Public-facing website for Ikan Bakar Kalianak. Served at `www.ikanbakarkalianak.store`.

---

## Development Commands

```bash
# From web_frontend/
npm run dev    # Vite dev server (port 3002 in Docker)
npm run build  # TypeScript + Vite production build
```
